Kmom04

Fångar olika event

Klick: elementet blir "selected" genom en outline

Dubbelklick: elementet blir mindre och försvinner

Tangent e: valda element blir rund

Tangent q: valda element blir större

Tangent w: valda element blir mindre

Tangent r: valda element ändrar färg

Tangent t: valda element kopierar sig själv

Tangent y: valda element raderas

Tangent u: alla element blir icke valda

Tangent i: alla element blir valda

Tangent p: element skapas med slumpmässig färg, form och position

Piltangent: flyttar valda element i lod- och vågrätt

Enter-tangenten: får valda element att "studsa"
